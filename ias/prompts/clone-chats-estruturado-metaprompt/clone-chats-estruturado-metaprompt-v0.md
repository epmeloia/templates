## 🧭 PASSO A PASSO — ABERTURA DO NOVO CHAT (SEM AMBIGUIDADE)

### **PASSO 1 — Abrir um novo chat**

* Abra **um chat totalmente novo** (não reutilize este).
* Não anexe nada ainda.
* Não escreva explicações.

---

### **PASSO 2 — PRIMEIRA mensagem do chat (obrigatoriamente texto puro)**

👉 **Cole EXATAMENTE estas 3 linhas como a PRIMEIRA mensagem**:

```
Use o arquivo `prompt-clonagem-continuidade-v10.2.md` como base absoluta de funcionamento deste chat.
Considere o arquivo `meta/continuity-backup.md` como fonte institucional imutavel e superior a qualquer instrucao isolada.
Aplique a LISTA MESTRA como template obrigatorio de controle e finalize todas as respostas com o Padrao de Resposta.
```

⚠️ **Regras críticas desse passo**

* Não anexe arquivo ainda
* Não coloque checklist
* Não coloque contexto
* Não explique nada

Esse passo **ativa o modo mental correto** do chat.

---

### **PASSO 3 — SEGUNDA mensagem: anexar os arquivos (sim, anexar)**

Agora sim, **anexe os arquivos físicos**, todos de uma vez:

* `prompt-clonagem-continuidade-v10.2.md`
* `meta/continuity-backup.md`

Depois de anexar, escreva **apenas uma linha**:

```
Arquivos anexados para referencia institucional.
```

Nada mais.

---

### **PASSO 4 — TERCEIRA mensagem: Confirmação de inicialização do chat**

Se quiser, **no novo chat**, você pode começar com:

```
Confirme que o chat iniciou corretamente.
```



---

### **PASSO 5 — QUARTA mensagem: comando simples de continuidade**

Agora você testa o sistema com algo **inofensivo e conhecido**, por exemplo:

```
# Gerar Somente Bloco 1
Data: 23/12/2025
"Antes de planejar o proximo passo, vale entender o caminho percorrido."
```

👉 A partir daqui:

* Gates funcionam
* Checklist aparece
* Padrao de resposta aparece
* Nenhuma inferencia fora do prompt ocorre

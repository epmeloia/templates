# Monitor de Temperatura

Aplicativo simples em Python para Windows 11 que mostra, em uma janela `tkinter`, informacoes de temperatura e estado da CPU e da GPU NVIDEA.

A janela fica sempre visivel sobre as outras janelas e atualiza as leituras a cada 2 segundos.

## Versao Atual

```txt
2026-06-04 v02.04
```

Esta versao do `README.md` foi atualizada para explicar como preparar um computador do zero, mesmo sem Python instalado.

## O Que A Janela Mostra

### CPU

- Media da temperatura dos nucleos.
- Nucleo maximo.
- CPU inteira.
- Estrangulamento termico.
- Limite de potencia.

### GPU NVIDEA

- Temperatura da GPU.
- Hot Spot.
- Temperatura da memoria.
- Uso da GPU.
- Clock da GPU.
- Limite termico.

## Cores

- Verde neon: temperatura dentro do limite.
- Amarelo: temperatura chegou no limite de atencao.
- Vermelho piscando: temperatura passou do limite critico.
- Azul neon: titulos e informacao de atualizacao.

Os limites padrao ja estao dentro do arquivo `monitor-temperatura.py`.

Se existir o arquivo `config-temperatura.json`, o app usa os limites dele.

## Arquivos Minimos

Para executar o monitor, os dois arquivos principais sao:

```txt
README.md
monitor-temperatura.py
```

O arquivo `README.md` explica o processo.

O arquivo `monitor-temperatura.py` executa a janela.

O HWiNFO precisa gerar o arquivo CSV de sensores neste caminho:

```txt
D:\_DEVOPS\Monitor_Temperatura\LogSensor\hwinfo-log.csv
```

Sem esse CSV, o app abre, mas nao consegue mostrar corretamente as leituras atuais da CPU e GPU.

## Versoes Usadas Neste Projeto

- Windows 11.
- Python 3.14.
- HWiNFO 64 v8.48-5990.
- Fan Control v267 foi usado apenas como referencia inicial de sensor.
- Desenvolvimento feito com auxilio de IA, usando ChatGPT/Codex.

## Passo 1 - Criar A Pasta Do Projeto

Crie esta pasta no Windows:

```txt
D:\_DEVOPS\Monitor_Temperatura
```

Dentro dela, coloque:

```txt
README.md
monitor-temperatura.py
```

Crie tambem a pasta:

```txt
D:\_DEVOPS\Monitor_Temperatura\LogSensor
```

## Passo 2 - Instalar O Python

Se o Python ainda nao estiver instalado:

1. Abra o navegador.
2. Acesse:

```txt
https://www.python.org/downloads/windows/
```

3. Baixe o instalador do Python para Windows 64-bit.
4. Execute o instalador.
5. Marque a opcao:

```txt
Add python.exe to PATH
```

6. Clique em instalar.

Depois, abra o PowerShell e teste:

```powershell
py -3 --version
```

Se aparecer a versao do Python, esta etapa esta pronta.

## Passo 3 - Instalar O HWiNFO

O Python nao le diretamente todos os sensores de temperatura do Windows.

Por isso, este projeto usa o HWiNFO para ler os sensores e gravar um CSV.

1. Abra o navegador.
2. Acesse:

```txt
https://www.hwinfo.com/download/
```

3. Baixe o instalador do HWiNFO 64.
4. Instale o programa.
5. Abra o HWiNFO.
6. Escolha o modo:

```txt
Apenas Sensores
```

## Passo 4 - Configurar O Log CSV Do HWiNFO

Na tela de sensores do HWiNFO:

1. Clique no botao de log.
2. Escolha a pasta:

```txt
D:\_DEVOPS\Monitor_Temperatura\LogSensor
```

3. Use o nome:

```txt
hwinfo-log.csv
```

O caminho final precisa ficar assim:

```txt
D:\_DEVOPS\Monitor_Temperatura\LogSensor\hwinfo-log.csv
```

Deixe o HWiNFO aberto enquanto usar o monitor.

## Passo 5 - Executar O Monitor

Abra o PowerShell e execute:

```powershell
cd "D:\_DEVOPS\Monitor_Temperatura"; py -3 .\monitor-temperatura.py
```

A janela do monitor deve abrir.

Se algo der erro, o PowerShell vai mostrar a mensagem.

## Como Confirmar Que Esta Funcionando

O app deve mostrar:

```txt
Fonte: HWiNFO CSV
```

E tambem:

```txt
Atualizado a cada 2 segundos.
```

Se aparecer `--`, confira:

- O HWiNFO esta aberto.
- O log CSV esta ativo.
- O arquivo `hwinfo-log.csv` existe na pasta `LogSensor`.
- O nome do arquivo esta exatamente `hwinfo-log.csv`.

## Sobre Dependencias Python

Para a leitura principal via HWiNFO CSV, o app usa bibliotecas padrao do Python.

O arquivo `requirements.txt` existe para tentativas e caminhos alternativos de leitura de sensor.

Se quiser instalar essas dependencias opcionais, execute:

```powershell
cd "D:\_DEVOPS\Monitor_Temperatura"; py -3 -m pip install -r requirements.txt
```

DECISOES.md

## Observacoes Importantes

- Mantenha o HWiNFO aberto enquanto o monitor estiver em uso.
- Se o CSV parar de atualizar, a janela pode continuar mostrando o ultimo valor gravado.
- Se a GPU ou CPU nao aparecerem, verifique se os sensores estao visiveis no HWiNFO.
- O texto `GPU NVIDEA` foi mantido conforme o padrao visual definido no projeto.

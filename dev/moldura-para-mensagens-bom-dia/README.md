## ğŸ§  Arquitetura Central â€” Sistema por Blocos

### ğŸ”¹ Bloco 1 â€” Imagem Principal

**Funcao**
- Comunicar a mensagem principal do dia.

**Conteudo exclusivo**

## ğŸš¦ Fluxo Obrigatorio (3 Etapas)

2. **Continuidade**

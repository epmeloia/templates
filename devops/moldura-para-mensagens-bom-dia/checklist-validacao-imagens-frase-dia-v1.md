# CHECKLIST DE VALIDAÇÃO (PARA USO ANTES DE ENTREGAR)

Use **SIM / NAO**.
Se houver **qualquer NAO**, a imagem **nao deve ser entregue**.

---

### ✅ Checklist Final — Imagem da Frase do Dia

**Interpretação**

* [ ] A imagem nasce da ideia da frase, nao da decoracao?
* [ ] A cena representa um momento especifico, nao generico?
* [ ] A imagem so faria sentido para este dia?

**Ambiente**

* [ ] O ambiente e diferente do dia anterior?
* [ ] O tipo de espaco mudou (interior/exterior, trabalho/descanso)?
* [ ] O horario ou clima narrativo mudou?

**Narrativa**

* [ ] Algo esta acontecendo na cena?
* [ ] Existe acao, reacao ou intencao?
* [ ] Os objetos interagem entre si?

**Estilo**

* [ ] E claramente Disney/Pixar?
* [ ] Nao parece fotografia?
* [ ] Nao parece render realista?

**Personagens**

* [ ] Nao ha humanos realistas?
* [ ] Objetos estao animados de forma suave?
* [ ] Nenhum personagem parece apenas “posar”?

**Texto**

* [ ] O texto e o foco principal?
* [ ] A data esta correta e bem posicionada?
* [ ] O banner e diferente do dia anterior?

**Climas**

* [ ] Natal/Ano Novo aparecem apenas como easter eggs?
* [ ] O clima nao domina a mensagem?

**Comparacao**

* [ ] A imagem nao se parece com a de ontem?
* [ ] A composicao nao parece reciclada?
* [ ] Se eu trocar a frase, a imagem perde o sentido?

# Monitor de Temperatura

Aplicativo simples em Python para Windows 11 que mostra, em uma janela `tkinter`, informações de temperatura e estado da CPU e da GPU NVIDEA.

A janela fica sempre visível sobre as outras janelas e atualiza as leituras a cada 2 segundos.

## Versão Atual

```txt
2026-06-04 v02.06
```

Esta versão do `README.md` foi atualizada para explicar como preparar um computador do zero, mesmo sem Python instalado.

## O Que A Janela Mostra

### CPU

- Média da temperatura dos núcleos.
- Núcleo máximo.
- CPU inteira.
- Estrangulamento térmico.
- Limite de potência.

### GPU NVIDEA

- Temperatura da GPU.
- Hot Spot.
- Temperatura da memória.
- Uso da GPU.
- Clock da GPU.
- Limite térmico.

## Cores

- Verde neon: temperatura dentro do limite.
- Amarelo: temperatura chegou no limite de atenção.
- Vermelho piscando: temperatura passou do limite crítico.
- Azul neon: títulos e informação de atualização.

Os limites padrão já estão dentro do arquivo `monitor-temperatura.py`.

O arquivo `config-temperatura.json` é usado para o usuário alterar os limites sem mexer no Python.

O app lê esse JSON sempre que abre.

Se o JSON não existir, o app cria um novo com os valores padrão.

Se o JSON estiver totalmente inválido, o app recria o arquivo inteiro com os valores padrão.

Se o JSON estiver parcialmente inválido, o app mantém as partes válidas e corrige as partes inválidas com os valores padrão.

## Arquivos Mínimos

Para executar o monitor, os dois arquivos principais são:

```txt
README.md
monitor-temperatura.py
```

O arquivo `README.md` explica o processo.

O arquivo `monitor-temperatura.py` executa a janela.

O arquivo `config-temperatura.json` é recomendado para configurar os limites de alerta.

Se ele não existir, o `monitor-temperatura.py` cria esse arquivo automaticamente.

O HWiNFO precisa gerar o arquivo CSV de sensores neste caminho:

```txt
D:\_DEVOPS\Monitor_Temperatura\LogSensor\hwinfo-log.csv
```

Sem esse CSV, o app abre, mas não consegue mostrar corretamente as leituras atuais da CPU e GPU.

## Passo 1 - Criar A Pasta Do Projeto

Crie esta pasta no Windows:

```txt
D:\_DEVOPS\Monitor_Temperatura
```

Dentro dela, coloque:

```txt
README.md
monitor-temperatura.py
```

Crie também a pasta:

```txt
D:\_DEVOPS\Monitor_Temperatura\LogSensor
```

## Passo 2 - Instalar O Python

Se o Python ainda não estiver instalado:

1. Abra o navegador.
2. Acesse:

```txt
https://www.python.org/downloads/windows/
```

3. Baixe o instalador do Python para Windows 64-bit.
4. Execute o instalador.
5. Marque a opção:

```txt
Add python.exe to PATH
```

6. Clique em instalar.

Depois, abra o PowerShell e teste:

```powershell
py -3 --version
```

Se aparecer a versão do Python, esta etapa está pronta.

## Passo 3 - Instalar O HWiNFO

O Python não lê diretamente todos os sensores de temperatura do Windows.

Por isso, este projeto usa o HWiNFO para ler os sensores e gravar um CSV.

1. Abra o navegador.
2. Acesse:

```txt
https://www.hwinfo.com/download/
```

3. Baixe o instalador do HWiNFO 64.
4. Instale o programa.
5. Abra o HWiNFO.
6. Escolha o modo:

```txt
Apenas Sensores
```

## Passo 4 - Configurar O Log CSV Do HWiNFO

Na tela de sensores do HWiNFO:

1. Clique no botão de log.
2. Escolha a pasta:

```txt
D:\_DEVOPS\Monitor_Temperatura\LogSensor
```

3. Use o nome:

```txt
hwinfo-log.csv
```

O caminho final precisa ficar assim:

```txt
D:\_DEVOPS\Monitor_Temperatura\LogSensor\hwinfo-log.csv
```

Deixe o HWiNFO aberto enquanto usar o monitor.

## Passo 5 - Executar O Monitor

Abra o PowerShell e execute:

```powershell
cd "D:\_DEVOPS\Monitor_Temperatura"; py -3 .\monitor-temperatura.py
```

A janela do monitor deve abrir.

Se algo der erro, o PowerShell vai mostrar a mensagem.

## Como Confirmar Que Está Funcionando

O app deve mostrar:

```txt
Fonte: HWiNFO CSV
```

E também:

```txt
Atualizado a cada 2 segundos.
```

Se aparecer `--`, confira:

- O HWiNFO está aberto.
- O log CSV está ativo.
- O arquivo `hwinfo-log.csv` existe na pasta `LogSensor`.
- O nome do arquivo está exatamente `hwinfo-log.csv`.

## Sobre Dependências Python

Para a leitura principal via HWiNFO CSV, o app usa bibliotecas padrão do Python.

O arquivo `requirements.txt` existe para tentativas e caminhos alternativos de leitura de sensor.

Se quiser instalar essas dependências opcionais, execute:

```powershell
cd "D:\_DEVOPS\Monitor_Temperatura"; py -3 -m pip install -r requirements.txt
```

## Observações Importantes

- Mantenha o HWiNFO aberto enquanto o monitor estiver em uso.
- Se o CSV parar de atualizar, a janela pode continuar mostrando o último valor gravado.
- Se a GPU ou CPU não aparecerem, verifique se os sensores estão visíveis no HWiNFO.
- O texto `GPU NVIDEA` foi mantido conforme o padrão visual definido no projeto.
